Help-Events list

This page allows you to manage events within the system. Here are the available actions:

  • Add New Event: Click on the "Add New" button to add a new event. This will redirect you to the form where you can enter the details for a new event.
  • Export to JSON: Click on the "Export to JSON" button to export the event details to a JSON file.The file will be downloaded and saved to your local device.
  • View Events List: The table below displays all the services currently in the system, including their name, date, time, description, type and location.
  • Edit Event: To edit an event, click on the "Edit" button next to the event you want to modify. This will open a form pre-filled with the event's current information.
  • Delete Event: To delete an event, click on the "Delete" button next to the corresponding event. A confirmation dialog may appear to ensure you want to proceed with the deletion. Note that this action is irreversible.
