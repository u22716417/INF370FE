Help- Check In

This page allows you to check in event attendees on event day.

  • Enter the ticket barcode either by manually typing it on the input box or by using the keyboard/touchpad on screen
  • Click on the "Check in"button to check in the attendee. A successful check-in client message will appear
  • Please note that you won't be able to check in the same ticket twice. An error message will appear.
