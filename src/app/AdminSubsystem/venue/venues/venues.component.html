Help-Venues list

This page allows you to manage venues within the system. Here are the available actions:

  • Add New Venue: Click on the "Add New" button to add a new venue. This will redirect you to the form where you can enter the details for a new venue.
  • Export to JSON: Use the "Export to JSON" button to export the list of venues as a JSON file. This is useful for data backup or integration purposes.
  • View Venues List: The table below displays all the venues currently in the system, including their name, location, and contact person.
  • Edit Venue: To edit a venue, click on the "Edit" button next to the venue you want to modify. This will open a form pre-filled with the venue's current information.
  • Delete Venue: To delete a venue, click on the "Delete" button next to the corresponding venue. A confirmation dialog may appear to ensure you want to proceed with the deletion. Note that this action is irreversible.
